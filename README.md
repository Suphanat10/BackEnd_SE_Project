
## ขั้นตอนการติดตั้ง project

Install my-project with npm

เข้า folder โปรเจค

```bash
    cd ./<project_folder>
```

ลง Package

```bash
  npm install
  หรือ
  npm i
```

การ run project

```bash
    node server.js
    หรือหากลง nodemon สามารถใช้
    ติดตั้ง npm i -g nodemon
    nodemon server.js
```
ทดลองเข้า http://localhost:8080/
